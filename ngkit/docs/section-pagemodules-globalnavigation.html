Standard Header Sticky Version

This header is seen on the standard non-project pages, after the normal navigation has disappeared from the screen. While in this example, it is not sticky, the header is should remain stuck to the top of the viewport.

NOTE Missing from the markup is the wrapper:

<div id="stickynav" class="ng-stickynav"></div>

This wrapper, combined with class ng-active triggers the hiding and stickyness of the header.

The JavaScript that must be added on the page:

$(document).ready(function() {
    var stickyNavTop = $('#defaultnav').offset().top + $('#defaultnav').height() + 40;

    var stickyNav = function(){
        var scrollTop = $.NGkit.$win.scrollTop();
        if (scrollTop > stickyNavTop) {
            $('#stickynav').addClass('ng-active');
        } else {
            $('#stickynav').removeClass('ng-active');
        }
    };

    stickyNav();

    $.NGkit.$doc.on('scrolling.ng.document', $.NGkit.Utils.debounce(function() {
        stickyNav();
    }, 50));

    $.NGkit.$win.on('resize orientationchange', $.NGkit.Utils.debounce(function() {
        stickyNav();
    }, 100));
});
