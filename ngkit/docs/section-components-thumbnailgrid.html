Thumbnail Grid

Options:

FieldRequired?Description
TitleNoThis appears at the top of the panel, above all the thumbnails
DescriptionNoThis text appears at the top of the panel, above all the thumbnails, but below the title.
WidthYesThis indicates how wide to make the overall module.
Thumbnail SizeYesThis size of the thumbnail images. This combined with the width field dictates how many thumbnails go across the module.
Display TypeNoOptionally used in some sites to provide additional styling on the module.
Action TextNoUsed with Action Link to provide a way to customize a link to additional content. If Action Link is not blank, this field defaults to "More {{ title }} >>".
Action LinkNoThis is a link to find more of this content. It appears at the bottom of the module, after all the thumbnails.

This uses the .ng-panel-themed modifier.

Thumbnail size depends on the module width and number of thumbnails across. Sizes given below are standardized, but may shrink a few pixels in some cases.
