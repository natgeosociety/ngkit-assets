Usage

To apply this component, add the .ng-form class to a form element. All form control elements are placed side by side within the next row.

NOTE In this example we used a button from the Button component. To add a top margin to form elements when they stack on smaller viewports, just add the data-uk-margin attribute from the Utility component to the parent element.

Example
I am the legend of the fieldset 
<form class="ng-form">
    <fieldset>
        <legend>I am the legend of the fieldset</legend>
        <input type="text" placeholder="username">
        <input type="password" placeholder="password">
        <select>
            <option>Option 1</option>
            <option>Option 2</option>
        </select>
        <button class="ng-button">Button</button>
        <label><input type="checkbox"> Checkbox</label>
    </fieldset>
</form>
Rows

Add the .ng-form-row class to a <div> element to stack form controls.

Example
I am the legend of the fieldset 
<form class="ng-form">
    <fieldset>
        <legend>I am the legend of the fieldset</legend>
        <div class="ng-form-row">
            <input type="text" placeholder="username">
        </div>
        <div class="ng-form-row">
            <input type="password" placeholder="password">
        </div>
        <div class="ng-form-row">
            <select>
                <option>Option 1</option>
                <option>Option 2</option>
            </select>
        </div>
        <div class="ng-form-row">
            <button>Button</button>
        </div>
        <div class="ng-form-row">
            <label><input type="checkbox"> Checkbox</label>
        </div>
    </fieldset>
</form>
States

Provide the user with basic information through feedback states on form controls.

Disabled state

Add the disabled attribute to a form control and it will be appear muted.

Example 
<form class="ng-form">
    <input type="text" placeholder="Don't touch this" disabled>
    <select disabled>
        <option>Option 1</option>
        <option>Option 2</option>
    </select>
</form>
Validation states

Add the .ng-form-danger or .ng-form-success class to a form control to notify the user whether or not the value is validated.

Example 
<form class="ng-form">
    <input type="text" placeholder="danger" class="ng-form-danger">
    <input type="text" placeholder="success" class="ng-form-success">
</form>
Control Modifiers
Blank form

Add the .ng-form-blank class to minimize the styling of form controls. The border and border style is used when the control is in focus.

Example 
<form class="ng-form">
    <input type="text" placeholder="text" class="ng-form-blank">
    <select class="ng-form-blank">
        <option>Option 1</option>
        <option>Option 2</option>
    </select>
</form>
Help text

Use the .ng-form-help-inline or .ng-form-help-block class to add inline or block level help text for the controls.

Example
The .ng-form-help-inline class creates spacing on the left.

The .ng-form-help-block class on a <p> tag creates an associated block of help text.

<form class="ng-form">
    <div class="ng-form-row">
        <input type="text" placeholder="text input">
        <span class="ng-form-help-inline">The
        <code>.ng-form-help-inline</code> class creates spacing on the left.</span>
    </div>
    <div class="ng-form-row">
        <textarea cols="30" rows="5" placeholder="textarea"></textarea>
        <p class="ng-form-help-block">The <code>.ng-form-help-block</code> class
        on a <code>&lt;p&gt;</code> tag creates an associated block of help text.</p>
    </div>
</form>
Size modifiers

Add the .ng-form-large or .ng-form-small class to an <input>, <select> or <textarea> element to make it smaller or larger.

Example 
<form class="ng-form">
    <div class="ng-form-row">
        <input type="text" placeholder="form-large" class="ng-form-large">
    </div>
    <div class="ng-form-row">
        <input type="text" placeholder="default">
    </div>
    <div class="ng-form-row">
        <input type="text" placeholder="form-small" class="ng-form-small">
    </div>
</form>
Width modifiers

Add the ng-form-width-large, ng-form-width-medium, ng-form-width-small or ng-form-width-mini class to an <input>, <select> or <textarea> element to adjust its width.

Example 
<form class="ng-form">
    <div class="ng-form-row"><input type="text" placeholder="form-width-large" class="ng-form-width-large"></div>
    <div class="ng-form-row"><input type="text" placeholder="form-width-medium" class="ng-form-width-medium"></div>
    <div class="ng-form-row"><input type="text" placeholder="form-width-small" class="ng-form-width-small"></div>
    <div class="ng-form-row"><input type="text" placeholder="form-width-mini" class="ng-form-width-mini"></div>
</form>
Layout modifiers

There are two layout modifiers available: .ng-form-stacked and .ng-form-horizontal. Both require form controls to be wrapped with the .ng-form-row class in order to stack them. Labels need the .ng-form-label class and controls must be wrapped in the .ng-form-controls class.

<form class="ng-form ng-form-stacked">
    <div class="ng-form-row">
        <label class="ng-form-label" for="">...</label>
        <div class="ng-form-controls">...</div>
    </div>
</form>

Note Layout modifiers can also be added to a <fieldset> element. This means if you use fieldsets you can have different form layouts for each fieldset.
