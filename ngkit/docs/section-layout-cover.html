Cover

Expand images or videos to cover their entire container.

This component allows you to create fullscreen teasers using images, objects or even iframes. Regardless what kind of element, it will always be centered vertically and horizontally and cover its container without losing its proportions. You can also place additional content, like text or an image, on top of the image or video.

Usage

The Cover component is applied differently, depending on whether you are using a background image, an object or an iframe. The simplest way is to add the .ng-cover-background class to a <div> element with a background image.

Example 
<div class="ng-cover-background" style="height: 300px; background-image: url(http://lorempixel.com/600/400);"></div>
Video

To create a video that covers its parent container, add the .ng-cover-object class to a video. Wrap a container element around the video and add the .ng-cover class to clip the content.

Example 
<div class="ng-cover" style="height: 300px;">
    <video class="ng-cover-object" width="600" height="400" autoplay="" loop="" muted="" controls="">
        <source src="http://www.quirksmode.org/html5/videos/big_buck_bunny.mp4?test1" type="video/mp4">
        <source src="http://www.quirksmode.org/html5/videos/big_buck_bunny.ogv?test1" type="video/ogg">
    </video>
</div>
IFrame

JAVASCRIPT Requires core/cover.js

To apply the Cover component to an iframe, you just need to add the data-ng-cover attribute to the iframe tag. Now you only need to add the .ng-cover class to a container element around the iframe to clip the content.

Example 
<div class="ng-cover" style="height: 300px;">
    <iframe data-ng-cover="" src="http://www.youtube.com/embed/YE7VzlLtp-4?autoplay=1&amp;controls=0&amp;showinfo=0&amp;rel=0&amp;loop=1&amp;modestbranding=1&amp;wmode=transparent&amp;enablejsapi=1&amp;api=1" width="560" height="315" frameborder="0" allowfullscreen="" style="width: 665px; height: 375px;"></iframe>
</div>
Responsive

To add responsive behavior to your cover image, you need to add the .ng-invisible class to an <img> element and place it inside your cover element. That way it will adapt the responsive behavior of the image.

NOTE Adding the .ng-height-viewport class from the Utility component will stretch the height of the parent element to fill the whole viewport.

Example 
<div class="ng-cover-background" style="background-image: url(http://lorempixel.com/600/400);">
    <img class="ng-invisible" src="http://lorempixel.com/600/400" width="600" height="400" alt="Placeholder">
</div>
Video

To add the same behavior to a video, you also need to add the .ng-position-relative class to the cover container and the .ng-position-absolute class to the cover object. The same applies to iframes.

Example 
<div class="ng-cover ng-position-relative">
    <img class="ng-invisible" src="http://lorempixel.com/600/400" width="600" height="400" alt="">
    <video class="ng-cover-object ng-position-absolute" width="600" height="400" autoplay="" loop="" muted="" controls="">
        <source src="http://www.quirksmode.org/html5/videos/big_buck_bunny.mp4?test2" type="video/mp4">
        <source src="http://www.quirksmode.org/html5/videos/big_buck_bunny.ogv?test2" type="video/ogg">
    </video>
</div>
Position content

You can also position content absolutely on top of the covering element. To do so, just add the .ng-position-cover class from the Utility component to a container element after your image or video. If you want to center the content vertically and horizontally, just use the Flex component.

Example
Bazinga!
<div class="ng-cover-background ng-position-relative" style="height: 300px; background-image: url(http://lorempixel.com/600/400);">
    <img class="ng-invisible" src="http://lorempixel.com/600/400" width="600" height="400" alt="">
    <div class="ng-position-cover ng-flex ng-flex-center ng-flex-middle">
        <div style="background: rgba(42, 142, 183, 0.8); font-size: 50px; line-height: 75px; color: #fff;">Bazinga!</div>
    </div>
</div>
