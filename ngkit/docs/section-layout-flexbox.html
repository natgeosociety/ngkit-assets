Flexbox

Utilize the power of Flexbox to create a wide range of layouts.

This component uses Flexbox, a concept that is still quite new but extremely powerful for creating layouts.

Flexbox
Usage
Flex inline
Modifiers
Alignment Item dimensions Direction Item order Wrap
Flex and grid
Usage

To apply this component, just add the .ng-flex class to a <div> element. This will create the flex container. By default, all flex items will be aligned to the left and equally matched in height and width.

Example
Box
Box
Box
<div class="ng-flex">
    <div class="ng-width-1-4 ng-panel ng-panel-box">Box</div>
    <div class="ng-width-1-4 ng-panel ng-panel-box ng-margin-left">Box</div>
    <div class="ng-width-1-4 ng-panel ng-panel-box ng-margin-left" style="height: 100px;">Box</div>
</div>
Flex inline

By default, the flex container is displayed as a block element. To apply the behavior of an inline element while still laying out its content according to the flexbox model, use the .ng-flex-inline class instead of .ng-flex.

Modifiers

You can add a number of different classes to modify the flex behavior.

Alignment

These classes define the horizontal and vertical alignment of flex items and distribute the space between them.

Class Description
.ng-flex-center Add this class to center flex items horizontally.
.ng-flex-right Add this class to align flex items to the right.
.ng-flex-top Add this class to align flex items to the top.
.ng-flex-middle Add this class to vertically center flex items.
.ng-flex-bottom Add this class to align flex items to the bottom.
.ng-flex-space-between Add this class to distribute items evenly, with the first item at the beginning and last item at the end of the main axis.
.ng-flex-space-around Add this class to distribute items evenly with equal space on both sides of each item.
Example
Box
Box
Box
<div class="ng-flex ng-flex-middle ng-flex-space-between">
   <div class="ng-width-1-4 ng-panel ng-panel-box" style="height: 50px;">Box</div>
   <div class="ng-width-1-4 ng-panel ng-panel-box" style="height: 70px;">Box</div>
   <div class="ng-width-1-4 ng-panel ng-panel-box" style="height: 100px;">Box</div>
</div>
Item dimensions

To determine how much space a flex item should take up, add one of the following classes to the item.

Class Description
.ng-flex-item-none The box's size is determined by its content.
.ng-flex-item-auto The space is allocated considering the item's content.
.ng-flex-item-1 The space is allocated solely based on flex.
Direction

These classes define the axis flex items placed on and their direction. By default, items run horizontally from left to right.

Class Description
.ng-flex-row-reverse Add this class to lay out flex items from right to left.
.ng-flex-column Add this class to lay out flex items as vertical columns.
.ng-flex-column-reverse Add this class to lay out flex items from bottom to top.
Example
1
2
3
<div class="ng-flex ng-flex-column ng-flex-column-reverse">
   <div class="ng-width-1-4 ng-panel ng-panel-box">1</div>
   <div class="ng-width-1-4 ng-panel ng-panel-box ng-margin-bottom">2</div>
   <div class="ng-width-1-4 ng-panel ng-panel-box ng-margin-bottom">3</div>
</div>
Item order

By default, flex items are laid out according to the source order. To display a certain item as the first or last one, just add one of these classes.

Class Description
.ng-flex-order-first Displays the item as the first one.
.ng-flex-order-last Displays the item as the last one.
.ng-flex-order-first-small
.ng-flex-order-last-small		 Affects device widths of 480px and higher.
.ng-flex-order-first-medium
.ng-flex-order-last-medium		 Affects device widths of 768px and higher.
.ng-flex-order-first-large
.ng-flex-order-last-large		 Affects device widths of 960px and higher.
.ng-flex-order-first-xlarge
.ng-flex-order-last-xlarge		 Affects device widths of 1220px and higher.
Example
1
2
3
<div class="ng-flex">
   <div class="ng-width-1-4 ng-panel ng-panel-box ng-margin-left">1</div>
   <div class="ng-width-1-4 ng-panel ng-panel-box ng-margin-left">2</div>
   <div class="ng-width-1-4 ng-panel ng-panel-box ng-flex-order-first">3</div>
</div>
Wrap

By default, flex items fit themselves into one row. Add the .ng-flex-wrap class, so that the items wrap into another row when they no longer fit the viewport. To change the items' direction so that they run from right to left, add the .ng-flex-wrap-reverse class. The following classes modify the alignment of wrapping flex items. To force the flex items into one row, just add the .ng-flex-nowrap class.

Class Description
.ng-flex-wrap-top Add this class to align multirow flex items to the top.
.ng-flex-wrap-middle Add this class to vertically center multirow flex items.
.ng-flex-wrap-bottom Add this class to align multiline flex items to the bottom.
.ng-flex-wrap-space-between Add this class to distribute item rows evenly, with the first row at the top and last row at the bottom of the container.
.ng-flex-wrap-space-around Add this class to distribute rows evenly with equal space at the top and bottom of each row.
Example
1
2
3
4
5
6
<div class="ng-flex ng-flex-wrap ng-flex-wrap-reverse ng-flex-wrap-space-around" style="height: 250px; background: #fafafa;">
    <div class="ng-width-1-3 ng-panel ng-panel-box ng-panel-box-primary">1</div>
    <div class="ng-width-1-2 ng-panel ng-panel-box ng-panel-box-primary ng-margin-left">2</div>
    <div class="ng-width-1-3 ng-panel ng-panel-box ng-panel-box-primary">3</div>
    <div class="ng-width-1-3 ng-panel ng-panel-box ng-panel-box-primary ng-margin-left">4</div>
    <div class="ng-width-1-2 ng-panel ng-panel-box ng-panel-box-primary">5</div>
    <div class="ng-width-1-3 ng-panel ng-panel-box ng-panel-box-primary ng-margin-left">6</div>
</div>
