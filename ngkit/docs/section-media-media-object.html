Media object and text wrapping

By default the textual content doesn't wrap around the image. You can change the default using the variable $media-object-wrap-default. You can add the .ng-wrap or .ng-nowrap class to the .ng-media-body class to reverse the default.

Media Object Variables
$media-object-wrap-defaultfalse
