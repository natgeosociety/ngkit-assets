Ordered and unordered lists

Create an unordered list using the <ul> element and the <ol> element for ordered lists. The <li> element defines the list item.

Example

Unordered list:

  • Item 1
  • Item 2
    • Item 2-1
    • Item 2-2
      • Item 2-2-1
      • Item 2-2-2
  • Item 3
  • Item 4

Ordered list:

  1. Item 1
  2. Item 2
    1. Item 2-1
    2. Item 2-2
      1. Item 2-2-1
      2. Item 2-2-2
  3. Item 3
  4. Item 4
<p><strong>Unordered list:</strong></p>
<ul>
    <li>Item 1</li>
    <li>Item 2
        <ul>
            <li>Item 2-1</li>
            <li>Item 2-2
                <ul>
                    <li>Item 2-2-1</li>
                    <li>Item 2-2-2
                </ul>
            </li>
        </ul>
    </li>
    <li>Item 3</li>
    <li>Item 4</li>
</ul>
<p><strong>Ordered list:</strong></p>
<ol>
    <li>Item 1</li>
    <li>Item 2
        <ol>
            <li>Item 2-1</li>
            <li>Item 2-2
                <ol>
                    <li>Item 2-2-1</li>
                    <li>Item 2-2-2
                </ol>
            </li>
        </ol>
    </li>
    <li>Item 3</li>
    <li>Item 4</li>
</ol>
Description lists

Create a description list using the <dl> element. Use <dt> to define the term and <dd> for the description.

Example
Description lists
A description list defines terms and their corresponding descriptions.
This is a term
This is a description.
This is a term
This is a description.
<dl>
    <dt>Description lists</dt>
    <dd>A description list defines terms and their corresponding descriptions.</dd>
    <dt>This is a term</dt>
    <dd>This is a description.</dd>
    <dt>This is a term</dt>
    <dd>This is a description.</dd>
</dl>
Usage

To apply this component, add the .ng-list class to an unordered or ordered list. The list will now display without any spacing or list-style.

Example
  • Item 1
  • Item 2
    • Subitem 1
    • Subitem 2
  • Item 3
<ul class="ng-list">
    <li>Item 1</li>
    <li>Item 2
    <ul><li>Subitem 1</li>
        <li>Subitem 2</li>
    </ul></li>
    <li>Item 3</li>
</ul>

Tree list

When you want to display hierarchical information with headings and subheads.

The top ul tag gets the class "ng-list-tree". An li tag gets the class "ng-list-treebranch" if it has children and should be marked as a heading.

If an li element has no children, use the class "ng-list-treeleaf".

Example
  • Geography
    • Physical Geography
  • Science
    • Biological and life sciences
  • Critical Thinking Skills
    • Analyzing
    • Understanding
  • Geographic Skills
    • Analyzing Geographic Information
    • Organizing Geographic Information 
<ul class="ng-list-tree">
  <li class="ng-list-treebranch">
    Geography
    <ul>
      <li class="ng-list-treeleaf">
        Physical Geography
      </li>
    </ul>
  </li>
  <li class="ng-list-treebranch">
    Science
    <ul>
      <li class="ng-list-treeleaf">
        Biological and life sciences
      </li>
    </ul>
  </li><li class="ng-list-treebranch">
    Critical Thinking Skills
    <ul>
      <li class="ng-list-treeleaf">
        Analyzing
      </li><li class="ng-list-treeleaf">
        Understanding
      </li>
    </ul>
  </li><li class="ng-list-treebranch">
    Geographic Skills
    <ul>
      <li class="ng-list-treeleaf">
        Analyzing Geographic Information
      </li><li class="ng-list-treeleaf">
        Organizing Geographic Information
      </li>
    </ul>
  </li>
</ul>
