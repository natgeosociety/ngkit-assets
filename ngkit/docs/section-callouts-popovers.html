Javascript
VariableDefaultMeaning
animationtrueDisplay the popover with animation.
placement'auto right'How to position the popover - top | bottom | left | right | auto. When "auto" is specified, it will dynamically reorient the popover. For example, if placement is "auto left", the popover will display to the left when possible, otherwise it will display right.
markup
'<div class="ng-popover">
   <div class="ng-arrow"></div>
   <div class="ng-popover-content"></div>
</div>'
The markup of the shell of the popover.
contentTarget'.ng-popover-content'The jQuery selector for the container within the markup in which the content is placed.
template'{{value}}'The template to use which formats the value attribute. If you wish to put the template in a <script> tag. Give the tag an ID like mytemplate and set the template to #mytemplate.
source''Array | URL string | function If it is a function, it is called with the callback function that the function should call with the data. this will be the Popover object. If it is an Array, It expects an array of objects with a value attribute. If it is a string, it makes an AJAX connection using param, value, and datatype configuration options.
param''Sent with the AJAX request
value''Sent with the AJAX request
datatype'json'Used with AJAX requests, set to jsonp for cross-domain requests.
trigger'hover focus'A space-delimited string indicating how popover is triggered - click | hover | focus | manual
delay0The delay in showing and hiding the popover in milliseconds. Does not apply to manual trigger type. If a number is supplied, delay is applied to both hide/show. Object structure is: delay: { show: 500, hide: 100 }
containerfalse
rendererfalse
viewport{selector: 'body',padding: 0}
