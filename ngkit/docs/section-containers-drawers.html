Drawer

Create a smooth sidebar that slides in and out of an element.

Drawer
Usage
Flipping the drawer
Vertical drawers
Tab Trigger
Alt Tab Trigger
Usage

There are two components: the drawer and the trigger

The drawer is the definition of how the drawer interacts with its parent, e.g. the closed width, the height, if it is initially open.

A drawer can have multiple triggers. Triggers tell the drawer to open or close.

If the drawer is partially open (closedWith > 0), the drawer itself can act as a trigger, opening on mouse over.

ClassDescription
.ng-drawerAdd this class to a <div> element to create the drawer container. This element also needs an id so the triggers can target it. The data-ng-drawer attribute is used to define the drawer's attributes.
.ng-drawer-barAdd this class to a child <div> element to hold the contents of the drawer.

Drawer JavaScript options

OptionDefaultDescription
initiallyOpenfalseThe drawer should start open
minWidth0The minimum width of the drawer in its opened state.
maxWidthnullThe maximum width of the drawer in its opened state. If null, it is set to its parent's width.
closedWidth0The width of the drawer in its closed state.
openOnHoverfalseOpen the drawer when the mouse hovers over the open part (requires closedWidth > 0) and closing automatically.

Drawer Triggers

You can use any element to toggle drawer; They only need the data-ng-drawer-trigger attribute. An <a> element's href should contain the drawer's id. Other elements, like buttons, just add the data-ng-drawertrigger="{target:'#ID'}" attribute to target the id of the drawer.

OptionDefaultDescription
target''The drawer's ID to trigger. An anchor's href attribute will set this if not set explicitly.
action'toggle'How this trigger should work. Choices are: open, close, toggle
