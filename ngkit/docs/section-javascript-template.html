Basic Templating

NGkit contains a very simple templating system. You can access it via NGkit.Utils.template.

JAVASCRIPT Requires cocore.js

MarkupMeaning
{{~variable}} ... {{/variable}} Iterate over every object in an array. Within the block, $item is the value of the current array item.
{{:variable}} ... {{/variable}} Iterate over every property in an object. Within the block, $key is the name of the property, and $val is its value
{{#variable}} ... {{/variable}} If variable evaluates as true (see what evaluates as true) then execute the block.
{{^variable}} ... {{/variable}} If variable evaluates as false (see what evaluates as true) then execute the block.
{{!variable}} Insert the value of variable without escaping.
{{variable}} Insert the escaped value of variable.
Basic Templating
Example 
<div id="outputContainer"></div>
<script>
datasource = {items: [{
    "value": "Africa: Human Geography",
    "html_url": "/education/encyclopedia/africa-human-geography/",
    "title": "Africa: Human Geography",
    "thumbnail": "http://lorempixel.com/60/45"
}, {
    "value": "Africa: Physical Geography",
    "html_url": "http://lorempixel.com/60/45",
    "title": "Africa: Physical Geography",
    "thumbnail": "http://lorempixel.com/60/45"
}, {
    "value": "Africa: Resources",
    "html_url": "/education/encyclopedia/africa-resources/",
    "title": "Africa: Resources",
    "thumbnail": "http://lorempixel.com/60/45"
}]};
</script>
<script type="text/template" id="example-tmpl">
    <ul class="ng-list ng-list-striped">
        {{#items && items.length}}
            {{~items}}
            <li class="ng-media">
                {{#$item.thumbnail}}<a href="{{$item.html_url}}" class="ng-media-img">
                    <img src="{{$item.thumbnail}}" width="60" height="45">
                </a>{{/$item.thumbnail}}
                <div class="ng-media-body">
                    <h4><a href="{{$item.html_url}}">{{$item.title}}</a></h4>
                </div>
            </li>
            {{/items}}
        {{/end}}
        {{^items.length}}
            <li>No results found.</li>
        {{/items.length}}
    </ul>
</script>
<script>
$(document).ready(function() {
    var tmpl = NGkit.Utils.template($('#example-tmpl').html(), datasource);
    $('#outputContainer').html(tmpl);
});
</script>
