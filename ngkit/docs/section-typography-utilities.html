Size modifiers
.ng-text-smallAdd this to decrease the font size
.ng-text-largeAdd this to increase the font size
Example

This class makes this small.

This is normal-sized text.

While this class makes this big.

<p class="ng-text-small">This class makes this small.</p>
<p>This is normal-sized text.</p>
<p class="ng-text-large">While this class makes this big.</p>
Weight modifiers
.ng-text-boldAdd this when you want the bold style without the semantics
.ng-text-normalAdd this to revert normally bold styled text to normal
Example

This text is styled bold. For when you want the bold style without the semantics.

This text is styled bold. This is normal. For when you want the normal style without the semantics.

<p class="ng-text-bold">This text is styled bold. For when you want the bold
style without the semantics.</p>
<h4>This text is styled bold. <span class="ng-text-normal">This is normal.</span>
For when you want the normal style without the semantics.</h4>

Example

<p class="ng-text-italic">This text is styled italic. For when you want the italic
style without the semantics.</p>

Alignment modifiers
Class Description
.ng-text-left Aligns text to the left.
.ng-text-right Aligns text to the right.
.ng-text-center Centers text horizontallly.
.ng-text-center-small Centers text horizontallly only on small devices.
.ng-text-center-medium Centers text horizontallly only on medium devices.
.ng-text-justify Justifies text.
