Usage

To apply this component, add the .ng-alert class to a block element.

Example
Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore. 
<div class="ng-alert">
    Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore.
</div>
Close button

To apply a close button, add the .ng-alert-close class to a <button> or <a> element inside the alert box. To enable the JavaScript for the close button, just add the data-ng-alert attribute to the alert box. We recommend adding the .ng-close class from the Close component to give the button a proper styling, though you can also use text or an image.

Example

Lorem ipsum dolor sit amet, consectetur adipisicing elit.

<div class="ng-alert" data-ng-alert="">
    <a href="" class="ng-alert-close ng-close"></a>
    <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit.</p>
</div>
Color modifiers

There are several color modifiers available. Just add one of the following classes to apply a different look.

Example

To indicate success or a positive message add the .ng-alert-success class.

To indicate a message containing a warning add the .ng-alert-warning class.

To indicate an important message add the .ng-alert-danger class.

For a different type of alert message add the .ng-alert-overlay class.

<div class="ng-alert ng-alert-success" data-ng-alert="">
    <a href="" class="ng-alert-close ng-close"></a>
    <p>To indicate success or a positive message add the <code>.ng-alert-success</code> class.</p>
</div>
<div class="ng-alert ng-alert-warning" data-ng-alert="">
    <a href="" class="ng-alert-close ng-close"></a>
    <p>To indicate a message containing a warning add the <code>.ng-alert-warning</code> class.</p>
</div>
<div class="ng-alert ng-alert-danger" data-ng-alert="">
    <a href="" class="ng-alert-close ng-close"></a>
    <p>To indicate an important message add the <code>.ng-alert-danger</code> class.</p>
</div>
<div class="ng-alert ng-alert-overlay" data-ng-alert="">
    <a href="" class="ng-alert-close ng-close"></a>
    <p>For a different type of alert message add the <code>.ng-alert-overlay</code> class.</p>
</div>
Size modifier

Add the .ng-alert-large class to increase the spacing in an alert box. This can be useful, if you want to use richer typography.

Example

Today's headline
