Slideshow

Create a responsive image or video slideshow with stunning transition effects, fullscreen mode and overlays.

JAVASCRIPT Requires components/slideshow.js, components/slideshow-fx.js

Usage

To create a slideshow, just add the .ng-slideshow class to a <ul> element and place your images inside the list items. In order to load the necessary JavaScript, you also need to add the data-ng-slideshow attribute.

NOTE To activate autoplay, just add the {autoplay:true} option to the data attribute.

Example 
<ul class="ng-slideshow" data-ng-slideshow="{autoplay:true}">
    <li><img src="http://lorempixel.com/800/400/city" width="800" height="400" alt=""></li>
    <li><img src="http://lorempixel.com/800/400/people" width="800" height="400" alt=""></li>
    <li><img src="http://lorempixel.com/800/400/animals" width="800" height="400" alt=""></li>
</ul>
Navigations

To navigate through your slides, just use the data-ng-slideshow-item attribute. To target the slides, you need to set the data attribute of every nav item to the number of the respective slideshow item. The elements with the data-ng-slideshow-item attribute need to be inside the data-ng-slideshow container.

<div data-ng-slideshow>
    <ul class="ng-slideshow">
        <li></li>
        <li></li>
    </ul>
    <ul>
        <li data-ng-slideshow-item="0"><a href="">...</a></li>
        <li data-ng-slideshow-item="1"><a href="">...</a></li>
    </ul>
</div>

Setting the attribute to next and previous will switch to the adjacent slides.

<div data-ng-slideshow>
    <ul class="ng-slideshow">
        <li></li>
        <li></li>
    </ul>
    <ul>
        <li data-ng-slideshow-item="next"><a href="">...</a></li>
        <li data-ng-slideshow-item="previous"><a href="">...</a></li>
    </ul>
</div>

Transitions

The Slideshow component allows you to add different animations to items when switching between them. All you need to do is add the animation parameter to the data-attribute and define the animation that you want to apply. Check the table below for an overview of the animations provided.

ClassDescription
fade The item fades in.
scroll The items scroll in.
scale The items scale up on out.
swipe The items swipe in and out.

To apply these more advanced transitions, the components/slideshow-fx.js file needs to be included in the document head.

ClassDescription
slice-down The items slide down in slices.
slice-up The items slide up in slices.
slice-up-down The sliced items slide in alterning directions.
fold The items are folded in.
puzzle The items are divided in squares that randomly fade in.
boxes The items are divided in squares that scale in diagonally from the top left corner.
boxes-reverse The items are divided in squares that scale in diagonally from the bottom right corner.
random-fx Different animations are applied randomly.

Ken Burns effect

The Ken Burns effect can also be applied to slideshow items. Just add the {kenburns:true} option to the data attribute. This effect can also be applied to videos.

For a custom animation duration use: {kenburns:'30s'} for 30 seconds.

Example 
<div class="ng-slidenav-position" data-ng-slideshow="{kenburns:true, autoplay:true}">
    <ul class="ng-slideshow">
        <li><img src="http://lorempixel.com/800/400/city" width="800" height="400" alt=""></li>
        <li><img src="http://lorempixel.com/800/400/people" width="800" height="400" alt=""></li>
        <li><img src="http://lorempixel.com/800/400/animals" width="800" height="400" alt=""></li>
    </ul>
    <a href="#" class="ng-slidenav ng-slidenav-contrast ng-slidenav-previous" data-ng-slideshow-item="previous" style="color: rgba(255,255,255,0.4)"></a>
    <a href="#" class="ng-slidenav ng-slidenav-contrast ng-slidenav-next" data-ng-slideshow-item="next" style="color: rgba(255,255,255,0.4)"></a>
</div>
Fullscreen slideshow

To create a fullscreen slideshow that stretches to fill the entire viewport, just add the .ng-slideshow-fullscreen class.

<ul class="ng-slideshow ng-slideshow-fullscreen" data-ng-slideshow>...</ul>
Overlays

You can also enhance slideshows with different overlays. Just add the .ng-overlay-panel to a <div> element inside the list item. Also add .ng-overlay-background and .ng-overlay-fade to show a background and a transition.

To toggle the overlay once a slide becomes active add the .ng-overlay-active to the list container. For more options, check out the Overlay component.

JavaScript Options

To initialize an element manually:

var slideshow = NGkit.slideshow(element, { });
Option Default Description
animation 'fade' Defines the preferred transition between items.
duration 500 Defines the transition duration.
height 'auto' Defines the slideshow height.
start 0 Defines the first slideshow item to be displayed.
autoplay false Defines whether or not the slideshow items should switch automatically.
pauseOnHover true Pause autoplay when hovering a slideshow.
autoplayInterval 7000 Defines the timespan between switching slideshow items.
videoautoplay true Defines whether or not a video starts automatically.
videomute true Defines whether or not a video is muted.
kenburns false Defines whether or not the Ken Burns effect is active. If kenburns is a numeric value, it will be used as the animation duration.
kenburnsanimations ng-animation-middle-left, ng-animation-top-right,
ng-animation-bottom-left, ng-animation-top-center,
ng-animation-bottom-right		 Animation series.
slices 15 Defines the number of slices, if a "Slice" transition is set.
Events
Name Parameter Description
show.ng.slideshow event, next slide, current slide On showing a new slide (after animation is finished)
beforeshow.ng.slideshow event, next slide, current slide Before showing a new slide (before animation is finished)
