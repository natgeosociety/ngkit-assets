Usage

To apply this component, just add the .ng-contrast class to a container or the element whose color you would like to adjust. Text, Buttons, the nav side from the Nav component, the subnav line and subnav pill from the Subnav component, Icons, default forms from the Form component as well as the list line from the List component will automatically change their appearence.

This comes in really handy when placing content inside differently colored blocks from the Block component or on top of images.
