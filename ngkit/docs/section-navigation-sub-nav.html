Subnav

Usage

To apply this component, add the .ng-subnav class to an <ul> element and nest <a> elements within the list. Use a <span> element instead to disable the item.

To apply this component, use the following classes. To align a subnav, for example to horizontally center it, you can use the Flex component.

Class Description
.ng-subnav Add this class to an <ul> element and nest <a> elements within the list.
.ng-active Add this class to a list item to apply an active state.
.ng-disabled Add this class to a list item to apply a disabled state.

    <li><a href="#">Item</a></li>
</ul>
Modifiers
Accordion Subnav

Using the .ng-subnav-accordion class with the data-ng-subnavaccordion attribute collapses every list item except the first to the width of the first child of the <a> tag in the list item. On hover, it expands that item. It always keeps one open.

Example
  • Email
  • Share
  • Print
  • Download
  • Email
    Share
    Print
    Download
<ul class="ng-subnav-accordion" data-ng-subnavaccordion>
    <li><a href="#"><i class="ng-icon-envelope-o ng-icon-fw"></i> Email</a></li>
    <li><a href="#"><i class="ng-icon-share-square-o ng-icon-fw"></i> Share</a></li>
    <li><a href="#"><i class="ng-icon-print ng-icon-fw"></i> Print</a></li>
    <li><a href="#"><i class="ng-icon-download ng-icon-fw"></i> Download</a></li>
</ul>
<ul class="ng-subnav-slider" data-ng-subnavaccordion="{initiallyOpen: -1, alwaysClose: true}">
    <li><a href="#"><i class="ng-icon-envelope-o ng-icon-fw"></i> Email</a><br/>
        <a href="#"><i class="ng-icon-share-square-o ng-icon-fw"></i> Share</a><br/>
        <a href="#"><i class="ng-icon-print ng-icon-fw"></i> Print</a><br/>
        <a href="#"><i class="ng-icon-download ng-icon-fw"></i> Download</a></li>
</ul>
Subnav with line separators

Add the .ng-subnav-line class to separate menu items with lines.

Example
  • Item
  • Item
  • Disabled
  • Item

    <li><a href="#">Item</a></li>

    <li><a href="#">Item</a></li>
</ul>
Subnav pill

Add the .ng-subnav-pill class to highlight the active menu item with a background. To apply an active state to a menu item, just add the .ng-active class.

Example
  • Item
  • Item
  • Disabled
  • Item
<ul class="ng-subnav ng-subnav-pill">
    <li class="ng-active"><a href="#">Item</a></li>
    <li><a href="#">Item</a></li>
    <li><span>Disabled</span></li>
    <li><a href="#">Item</a></li>
</ul>
Subnav with dropdowns

This is an example of how to use a subnav with the Dropdown component.

Example
  • Active
  • Item
  • Disabled
  • More
    • Item
    • Another item
    • Header
    • Item
    • Another item
    • Separated item
<ul class="ng-subnav ng-subnav-pill">
    <li class="ng-active"><a href="#">Active</a></li>
    <li><a href="#">Item</a></li>
    <li><span>Disabled</span></li>
    <li data-ng-dropdown="{mode:'click'}">
        <a href="#">More <i class="ng-icon-caret-down"></i></a>
        <div class="ng-dropdown ng-dropdown-small">
            <ul class="ng-nav ng-nav-dropdown">
                <li><a href="#">Item</a></li>
                <li><a href="#">Another item</a></li>
                <li class="ng-nav-header">Header</li>
                <li><a href="#">Item</a></li>
                <li><a href="#">Another item</a></li>
                <li class="ng-nav-divider"></li>
                <li><a href="#">Separated item</a></li>
            </ul>
        </div>
    </li>
</ul>
