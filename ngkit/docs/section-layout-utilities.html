Container

Add the .ng-container class to a block element to give it a max-width and wrap the main content of your website. For large screens it applies a different max-width.

Centering

To center the container, use the .ng-container-center class. For any other block element, you additionally need to apply a width.

Example
Centered block element
<div class="ng-width-medium-1-2 ng-container-center">
    <div class="ng-panel ng-panel-box ng-panel-box-primary">Centered block element</div>
</div>
Clearing and floating

Floating is fundamental for creating all kinds of layouts. But floats need to be cleared or in the worst case, you might end up with a scrambled site. The following classes will help you to setup basic layouts.

Class Description
.ng-float-left Float the element to the left.
.ng-float-right Float the element to the right.
.ng-clearfix Add this class to a parent container to clear floats.
Example
Floated to the right
Floated to the left
<div class="ng-clearfix">
    <div class="ng-panel ng-panel-box-primary ng-width-medium-1-3 ng-float-right">Floated to the right</div>
    <div class="ng-panel ng-panel-box-primary ng-width-medium-1-3 ng-float-left">Floated to the left</div>
</div>
Alignment of images and objects

Align images or other elements with spacing between the text and the element.

Class Description
.ng-align-left Floats the element to the left and creates right and bottom margin.
.ng-align-right Floats the element to the right and creates left and bottom margin.
.ng-align-medium-left Only affects device widths of 768px and higher.
.ng-align-medium-right Only affects device widths of 768px and higher.
.ng-align-center Centers the element and creates bottom margin.

New block formatting context

Instead of using the .ng-clearfix class, you can create a new block formatting context to clear floats. Depending on your setup, you can evaluate which method is more suited.

ClassDescription
.ng-nbfcSets overflow to hidden to create a new block formatting context.
.ng-nbfc-altSets display to table-cell to create a new block formatting context.
Vertical alignment

To align objects vertically, you need to create a parent container to which the object itself will be aligned.

NOTE The object you align needs to have a width or max-width that is smaller than its parent container.

Class Description
.ng-vertical-align Add this class to the parent container. This container needs a specific height.
.ng-vertical-align-middle Add this class to the child element to center your content.
.ng-vertical-align-bottom Add this class to the child element to align your content to the bottom.
.ng-height-1-1 This helper class applies a height of 100%.
Example

Lorem ipsum dolor sit amet, consectetur adipisicing elit.

Lorem ipsum dolor sit amet, consectetur adipisicing elit.

<div class="ng-grid" data-ng-grid-margin="">
    <div class="ng-width-medium-1-2">
        <div class="ng-vertical-align ng-panel ng-panel-box" style="height: 200px;">
            <div class="ng-vertical-align-middle ng-width-medium-1-2">
                <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit.</p>
            </div>
        </div>
    </div>
    <div class="ng-width-medium-1-2">
        <div class="ng-vertical-align ng-panel ng-panel-box" style="height: 200px;">
            <div class="ng-vertical-align-bottom ng-width-medium-1-2">
                <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit.</p>
            </div>
        </div>
    </div>
</div>
Center horizontally

To center your content horizontally as well, add the .ng-text-center class to the parent and the .ng-container-center class to the child element. This is necessary because of the responsive behavior.

Center the entire page

The .ng-height-1-1 class comes in handy, if you want to extend the <html> and <body> elements to the full height of the page. This can be very useful when creating error pages.

<html class="ng-height-1-1">
    <body class="ng-height-1-1">
        <div class="ng-vertical-align">
            <div class="ng-vertical-align-middle">...</div>
        </div>
    </body>
</html>
Viewport Height

To create a container that fills the height of the entire viewport, for example for fullscreen image or video teasers, just add the .ng-height-viewport class.

Responsive objects

Images in NGkit adapt to the width of their parent container by default. If you want to apply the responsive behavior to media elements, like a video object, just add one of the following classes.

Class Description
.ng-responsive-width Adjusts the object's width according to its parent's width, keeping the original aspect ratio.
.ng-responsive-height Adjusts the object's height according to its parent's height, keeping the original aspect ratio.
Responsive height
Example 

<p style="height: 125px;">
    <img class="ng-responsive-height" src="http://lorempixel.com/400/200" width="400" height="200">
</p>

Inline SVG

SVGs or Scaleable Vector Graphics are really handy, for example to display a logo that remains crisp when scaling or that is animated. To be able to control your SVG via CSS, just add the data-ng-svg attribute to the image tag. This will turn your image into an inline SVG including all attributes, like IDs, classes, width and height which you can now easily target using CSS.

Auto margin

To add spacing to stacking elements, for example inline elements that wrap on smaller vieports, just add the data-ng-margin attribute to their parent container. It will automatically add the .ng-margin-small-top to the lower element.

NOTE By default, the data attribute adds the .ng-margin-small-top class to stacking elements. To apply a bigger margin, just add the {cls:'ng-margin-top'} option.

Example
