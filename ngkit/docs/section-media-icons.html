Icons

Place vector icons anywhere using an icon font.

This component uses a custom Icon Society icon font. Icon fonts are great, because they enable you to easily change color, size and more via CSS. They are scalable vector graphics, which means that they look great on high-resolution displays.

Basic Usage

To apply an icon, add any .ng-icon-* class to an <i> or <span> element. Et voil√†, you have a vector icon, which inherits size and color just like your text does.

Example
  • Uses the .ng-icon-download class
  • Uses the .ng-icon-user class
  • Uses the .ng-icon-sort class
  • An icon in a link
A button with an icon
<ul class="ng-list ng-list-line">
    <li><i class="ng-icon-download"></i> Uses the <code>.ng-icon-download</code> class</li>
    <li><i class="ng-icon-user"></i> Uses the <code>.ng-icon-user</code> class</li>
    <li><i class="ng-icon-sort"></i> Uses the <code>.ng-icon-sort</code> class</li>
    <li><a href="#"><i class="ng-icon-external-link"></i> An icon in a link</a></li>
</ul>
<a href="#" class="ng-button"><i class="ng-icon-envelope"></i> A button with an icon</a>

</div>

Size Modifiers

Add the .ng-icon-small, .ng-icon-medium or .ng-icon-large class to an icon to make the it larger.

Example

the default icon is the same size as the text

.ng-icon-small is 150% of the text size

.ng-icon-medium is 200% of the text size

.ng-icon-large is 250% of the text size

<p><i class="ng-icon-ngs"></i> the default icon is the same size as the text</p>
<p><i class="ng-icon-ngs ng-icon-small"></i> <code>.ng-icon-small</code> is 150% of the text size</p>
<p><i class="ng-icon-ngs ng-icon-medium"></i> <code>.ng-icon-medium</code> is 200% of the text size</p>
<p><i class="ng-icon-ngs ng-icon-large"></i> <code>.ng-icon-large</code> is 250% of the text size</p>
Spin Modifier

Use the ng-icon-spin class to get any icon to rotate.

Example 
<i class="ng-icon-clock-o ng-icon-spin"></i>
<i class="ng-icon-close ng-icon-spin"></i>
<i class="ng-icon-share ng-icon-spin"></i>
Icon hover

Use the modifier .ng-icon-hover class to use an anchor as an icon, applying a hover effect and removing the link underline.

Example 
<a href="#" class="ng-icon-small ng-icon-hover ng-icon-facebook"></a>
<a href="#" class="ng-icon-small ng-icon-hover ng-icon-twitter"></a>
<a href="#" class="ng-icon-small ng-icon-hover ng-icon-google-plus"></a>
<a href="#" class="ng-icon-small ng-icon-hover ng-icon-instagram"></a>

Justify icon

To add a fixed width to the icon and center it, add the .ng-icon-justify class. This is useful when using different icons in a list.

Example
  • Downloads
  • Library
  • Users
  • Favorites
<ul class="ng-list">
  <li><a href="#"><i class="ng-icon-download ng-icon-justify"></i> Downloads</a></li>
  <li><a href="#"><i class="ng-icon-book ng-icon-justify"></i> Library</a></li>
  <li><a href="#"><i class="ng-icon-user ng-icon-justify"></i> Users</a></li>
  <li><a href="#"><i class="ng-icon-heart ng-icon-justify"></i> Favorites</a></li>
</ul>
