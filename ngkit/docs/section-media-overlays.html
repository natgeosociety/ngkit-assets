Usage

This component is easily applied. To create a position context, add the .ng-overlay class to a container element around an image. Add the .ng-overlay-panel class to a child element to create the actual overlay panel. Usually you would use a <figure> and <figcaption> element for this.

Example 
<div class="ng-width-1-2">
    <figure class="ng-overlay">
        <img src="http://lorempixel.com/600/400" width="600" height="400" alt="">
        <figcaption class="ng-overlay-panel">
            <h3>Title</h3>
            <p>Lorem <a href="#">ipsum dolor</a> sit amet, consetetur sadipscing elitr.</p>
        </figcaption>
    </figure>
</div>
Toggle overlay on hover

By default, the overlay is always visible. To hide the overlay and display it on hover, add the .ng-overlay-hover class to the overlay container.

Example 
<div class="ng-width-1-2">
    <figure class="ng-overlay ng-overlay-hover">
        <img src="http://lorempixel.com/600/400" width="600" height="400" alt="">
        <figcaption class="ng-overlay-panel">
            <h3>Title</h3>
            <p>Lorem <a href="#">ipsum dolor</a> sit amet, consetetur sadipscing elitr.</p>
        </figcaption>
    </figure>
</div>
Toggle on active

To toggle an overlay when its parent container is in an active state, add the .ng-overlay-active class. This comes in handy, for example, in combination with the Slideshow component.

Overlay background

To give the overlay a background just add the .ng-overlay-background class to the overlay-panel container.

    </figure>
</div>

Overlay position

ClassDescription
.ng-overlay-top Add this class to align the overlay to the top.
.ng-overlay-bottom Add this class to align the overlay to the bottom.
.ng-overlay-left Add this class to align the overlay to the left.
.ng-overlay-right Add this class to align the overlay to the right.

Overlay transitions

Overlay panels and images can easily be animated. Just add one of the following classes to the overlay panel or <img> element. Note, that this only works in combination with the .ng-overlay-hover or .ng-overlay-active class.

Class Description
.ng-overlay-slide-top Add this class to the overlay panel to slide it in from the top.
.ng-overlay-slide-bottom Add this class to the overlay panel to slide it in from the bottom.
.ng-overlay-slide-left Add this class to the overlay panel to slide it in from the left.
.ng-overlay-slide-right Add this class to the overlay panel to slide it in from the right.
.ng-overlay-fade Add this class to the overlay panel or image to fade it in.
.ng-overlay-scale Add this class to the image to scale it up.
.ng-overlay-spin Add this class to the image to slightly rotate it to the right.
.ng-overlay-grayscale Add this class to the image to desaturate it and color it on hover.

NOTE If you'd like to animate the image, but want the overlay panel to be visible at all times, just add the .ng-ignore class to the panel. That way it will be displayed despite the .ng-overlay-hover class.

Example 
<ul class="ng-grid" data-ng-grid-margin="">
    <li class="ng-width-medium-1-3">
        <figure class="ng-overlay ng-overlay-hover">
            <img src="http://lorempixel.com/600/400" width="600" height="400" alt="Image">
            <figcaption class="ng-overlay-panel ng-overlay-background  ng-overlay-bottom ng-overlay-slide-bottom">
                <p>Lorem ipsum dolor sit amet.</p>
            </figcaption>
        </figure>
    </li>
    <li class="ng-width-medium-1-3">
        <figure class="ng-overlay ng-overlay-hover">
            <img class="ng-overlay-scale" src="http://lorempixel.com/600/400" width="600" height="400" alt="Image">
            <figcaption class="ng-overlay-panel ng-overlay-background ng-overlay-bottom ng-ignore">
                <p>Lorem ipsum dolor sit amet.</p>
            </figcaption>
        </figure>
    </li>
    <li class="ng-width-medium-1-3">
        <figure class="ng-overlay ng-overlay-hover">
            <img class="ng-overlay-spin" src="http://lorempixel.com/600/400" width="600" height="400" alt="Image">
            <figcaption class="ng-overlay-panel ng-overlay-background ng-overlay-bottom ng-overlay-slide-bottom">
                <p>Lorem ipsum dolor sit amet.</p>
            </figcaption>
        </figure>
    </li>
</ul>
Overlay anchor

To use the entire overlay as a link, just place an <a> element inside the overlay container and add the .ng-position-cover class from the Utility component.

Example 
<div class="ng-width-medium-1-2">
    <figure class="ng-overlay ng-overlay-hover">
        <img src="http://lorempixel.com/600/400" alt="Image">
        <figcaption class="ng-overlay-panel ng-overlay-fade ng-overlay-background">
            <h3>Anchor</h3>
            <p>Lorem ipsum dolor sit amet.</p>
        </figcaption>
        <a class="ng-position-cover" href="#"></a>
    </figure>
</div>
