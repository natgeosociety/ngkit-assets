Usage

The modal component consists of an overlay, a dialog and a close button.

Class Description
.ng-modal Add this class to a <div> element to create the dialog container and an overlay that blanks out the page. It is important to add an id to indicate the element for toggling.
.ng-modal-dialog Add this class to a child <div> element to create the dialog box.
.ng-modal-close Add this class to an <a> or <button> element to create a close button within the dialog box. We recommend adding the .ng-close class from the Close component to give the button a proper styling, though you can also use text or an image.

You can use any element to toggle a modal dialog. An <a> element needs to be linked to the modal's id. To enable the necessary JavaScript, add the data-ng-modal attribute. If you are using another element, like a button, just add the data-ng-modal="{target:'#ID'}" attribute to target the modal's id.

JavaScript options

By default, the modal closes automatically when clicking on the modal overlay. To prevent this from happening, just add the data-ng-modal="{target:'#ID',bgclose:false}" attribute.

Lightbox modifier

To create a lightbox-like modal dialog, just add the .ng-modal-dialog-lightbox class. This can be useful, if you want to use the modal as a lightbox for your images. The close button will adjust its position automatically to the dialog.

NOTE When creating a close button within the lightbox modifier, we also recommend adding the .ng-close-alt class from the Close component to give your button a styling that fits the lightbox modal.

Example 
<button class="ng-button" data-ng-modal="{target:'#modal-3'}">Open</button>
<div id="modal-3" class="ng-modal">
    <div class="ng-modal-dialog ng-modal-dialog-lightbox">
        <a href="" class="ng-modal-close ng-close ng-close-alt"></a>
        <img src="http://lorempixel.com/600/400" width="600" height="400" alt="">
    </div>
</div>
Modal blank

To reset all styling, like padding and margin, add the .ng-modal-dialog-blank class. This can be useful, if you want to create a fullscreen modal. In that case you also need to add the .ng-height-viewport class from the Utility component so that the modal extends to full viewport height.

Dialogs

The modal component also provides alternatives to the native dialogs: window.alert, window.confirm and window.prompt.

Alert Dialog 
NGkit.modal.alert('Attention!')
Example 
<button type="button" class="ng-button" onclick="NGkit.modal.alert('Attention!')">Alert</button>
Block UI 
(
    function(modal){
        modal = NGkit.modal.blockUI('Wait 5 sec...');
        setTimeout(function(){ modal.hide() }, 5000)
    }
)();
Example 
<button type="button" class="ng-button" onclick="(function(modal){ modal = NGkit.modal.blockUI('Wait 5 sec...'); setTimeout(function(){ modal.hide() }, 5000) })();">Block UI</button>
Confirm Dialog 
NGkit.modal.confirm('Are you sure?', function(){
    NGkit.modal.alert('Confirmed!');
});
Example 
<button type="button" class="ng-button" onclick="NGkit.modal.confirm('Are you sure?', function(){ NGkit.modal.alert('Confirmed!'); });">Confirm</button>
Prompt Dialog 
NGkit.modal.prompt('Name:', '', function(val){
    NGkit.modal.alert('Hello '+(val || 'Mr noname')+'!');
});
Example 
<button type="button" class="ng-button" onclick="NGkit.modal.prompt('Name:', '', function(val){ NGkit.modal.alert('Hello '+(val || 'Mr noname')+'!'); });">Prompt</button>
JavaScript

You can handle modal dialogs via raw javascript.

###Example

var modal = new $.NGkit.modal.Modal(".modalSelector");

if ( modal.isActive() ) {
    modal.hide();
} else {
    modal.show();
}
Events

The modal component triggers an ng.modal.show event every time a modal is opened and ng.modal.hide when a modal is closed.

Name Parameter Description
show.ng.modal event On modal show
hide.ng.modal event On modal hide

Example

$('.modalSelector').on({

    'show.ng.modal': function(){
        console.log("Modal is visible.");
    },

    'hide.ng.modal': function(){
        console.log("Element is not visible.");
    }
});
