Basic Usage

To apply this component, add the .ng-button class to an <a> or <button> element. Now you have created a button. Add the disabled attribute to a <button> element to disable the button.

Example
Link 
<a class="ng-button [modifier class]" href="">Link</a>
<button class="ng-button [modifier class]" type="button">Button</button>
:hover
Highlight the button when hovering
Link
:disabled
When disabled, the buttons's appears dimmed.
Link
Color modifiers

There are several color modifiers available. Just add one of the following classes to apply a different look.

Example

<a class="ng-button-disabled">This Button is Disabled</a>

Button Link Modifier

To deemphasize a button to look like a link while maintaining button behavior, add .ng-button-link.

Example
Link 
<a class="ng-button ng-button-link" href="">Link</a>
<button class="ng-button ng-button-link" type="button">Button</button>
<button class="ng-button ng-button-link" type="button" disabled>Disabled</button>
Style Modifiers

To deemphasize a button to have no styling while maintaining button behavior, add .ng-button-blank

Example
Link 
<a class="ng-button ng-button-blank" href="">Link</a>
<button class="ng-button ng-button-blank" type="button">Button</button>
<button class="ng-button ng-button-blank" type="button" disabled>Disabled</button>
Size modifiers

Add the .ng-button-mini, .ng-button-small or .ng-button-large class to a button to make it smaller or larger.

Example 
<button class="ng-button ng-button-mini" type="button">Mini button</button>
<button class="ng-button ng-button-small" type="button">Small button</button>
<button class="ng-button" type="button">Default button</button>
<button class="ng-button ng-button-large" type="button">Large button</button>
Button Group

To create a button group, add the .ng-button-group class to a <div> element around the buttons. That's it! No further markup needed.

Example
Link 
<div class="ng-button-group">
    <a class="ng-button" href="">Link</a>
    <button class="ng-button">Button</button>
    <button class="ng-button">Button</button>
</div>

JavaScript

JAVASCRIPT Requires core/button.js

You can toggle button states via JavaScript. Just add the data attibute data-ng-button.

Example 
<button class="ng-button ng-button-primary" type="button" data-ng-button="">Button</button>
Checkbox Buttons

Toggle between a group of buttons like a checkbox by wrapping a

element with the data attribute data-ng-button-checkbox around them. This can also be applied to a button group.

Example 
<div class="ng-margin">
    <div data-ng-button-checkbox="">
        <button class="ng-button">Button</button>
        <button class="ng-button">Button</button>
        <button class="ng-button">Button</button>
    </div>
</div>
<div class="ng-margin">
    <div class="ng-button-group" data-ng-button-checkbox="">
        <button class="ng-button">Button</button>
        <button class="ng-button">Button</button>
        <button class="ng-button">Button</button>
    </div>
</div>
Radio Buttons

Toggle between a group of buttons, like radio buttons, by wrapping a div element that uses data-ng-button-radio around them. This can also be applied to a button group.

Example 
<div class="ng-margin">
    <div data-ng-button-radio="">
        <button class="ng-button">Button</button>
        <button class="ng-button">Button</button>
        <button class="ng-button">Button</button>
    </div>
</div>
<div class="ng-margin">
    <div class="ng-button-group" data-ng-button-radio="">
        <button class="ng-button">Button</button>
        <button class="ng-button">Button</button>
        <button class="ng-button">Button</button>
    </div>
</div>
