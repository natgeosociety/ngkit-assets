Usage

To apply this component, just add the .ng-thumbnail class to an <img>, <a> or <figure> element.

Example 
<!-- This is an image as a thumbnail -->
<img class="ng-thumbnail" src="http://lorempixel.com/100/75" alt="hello kitty!">
<!-- This is an anchor as a thumbnail -->
<a class="ng-thumbnail" href="">
  <img src="http://lorempixel.com/100/75" alt="hello kitty!">
</a>
<!-- This is a figure as a thumbnail -->
<figure class="ng-thumbnail">
  <img src="http://lorempixel.com/100/75" alt="hello kitty!">
</figure>
Bordered thumbnail

If you want to add borders to the thumbnail boxes, Add a .ng-bordered class with the .ng-thumbnail class to apply a border.

Example 
<!-- This is an image as a thumbnail -->
<img class="ng-thumbnail ng-bordered" src="http://lorempixel.com/100/75" alt="hello kitty!">
<!-- This is an anchor as a thumbnail -->
<a class="ng-thumbnail ng-bordered" href="">
  <img src="http://lorempixel.com/100/75" alt="hello kitty!">
</a>
<!-- This is a figure as a thumbnail -->
<figure class="ng-thumbnail ng-bordered">
  <img src="http://lorempixel.com/100/75" alt="hello kitty!">
</figure>
Caption

Add the .ng-thumbnail-caption class to a <div> element to apply a text caption under the image.

Example
This is a <div> as a thumbnail with a caption
This is an <a> as a thumbnail with a caption
 
<!-- This is a div as a thumbnail with a caption -->
<div class="ng-thumbnail ng-bordered ng-thumbnail-small">
    <img src="http://lorempixel.com/200/100" alt="hello kitty!">
    <div class="ng-thumbnail-caption">This is a <code>&lt;div&gt;</code> as a thumbnail with a caption</div>
</div>
<!-- This is an anchor as a thumbnail with a caption -->
<a class="ng-thumbnail ng-bordered ng-thumbnail-small" href="">
    <img src="http://lorempixel.com/200/100" alt="hello kitty!">
    <div class="ng-thumbnail-caption">This is an <code>&lt;a&gt;</code> as a thumbnail with a caption</div>
</a>
<!-- This is a figure as a thumbnail with a caption -->
<figure class="ng-thumbnail ng-bordered ng-thumbnail-small">
    <img src="http://lorempixel.com/200/100" alt="hello kitty!">
    <figcaption class="ng-thumbnail-caption">This is a <code>&lt;figure&gt;</code> as a thumbnail with a caption</figcaption>
</figure>

Expanding

You can even scale a thumbnail beyond its proper size, so that it extends to the full width of its parent element. Just add the .ng-thumbnail-expand class.

Example
.ng-thumbnail-expand
<div class="ng-thumbnail ng-thumbnail-expand">
  <img src="http://lorempixel.com/600/400" alt="hello kitty!">
  <div class="ng-thumbnail-caption"><code>.ng-thumbnail-expand</code></div>
</div>

Image Links

To provide an Opacity Hover Effect to show that an Image is a clickable link. To apply, simply add the ng-image-link class to the <a> tag.

Example 
<a class="ng-image-link" href="#">
    <img src="http://lorempixel.com/400/300/" />
</a>
