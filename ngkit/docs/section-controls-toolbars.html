Usage

The Toolbar component consists of the toolbar itself and one or more navigations.

Class Description
.ng-toolbar Add this class to a <nav> element to define the Toolbar component.
.ng-toolbar-nav Add this class to a <ul> element to create a navigation. Use <a> elements as menu items within the list.
.ng-active Add this class to a list item to apply an active state.
Example 

<div class="ng-toolbar" data-ng-toolbar>
    <ul class="ng-toolbar-nav">
        <li><a href=""><i class="ng-icon-minus ng-icon-fw"></i></a></li>
        <li><span style="width: 150px"><input type="range" min="0" max="12" step="0.1" data-ng-slider></span></li>
        <li><a href=""><i class="ng-icon-plus ng-icon-fw"></i></a></li>
    </ul>
</div>
<br/>
<div class="ng-toolbar" data-ng-toolbar="{mode: 'radio'}">
    <ul class="ng-toolbar-nav">
        <li><a href=""><i class="ng-icon-arrows ng-icon-fw"></i></a></li>
        <li><a href=""><i class="ng-icon-eraser ng-icon-fw"></i></a></li>
        <li><a href=""><i class="ng-icon-map-marker ng-icon-fw"></i></a></li>
    </ul>
</div>
Toolbar subtitle

Add the .ng-toolbar-nav-subtitle class to an <a> element in the list item to indicate that it has a subtitle. Use a <div> element for the subtitle itself.

Example
  • Active
    Subtitle
  • Item
    Subtitle
  • Parent
    Subtitle
     
<div class="ng-toolbar">
    <ul class="ng-toolbar-nav">
        <li class="ng-active"><a href="" class="ng-toolbar-nav-subtitle">Active<div>Subtitle</div></a></li>
        <li><a href="" class="ng-toolbar-nav-subtitle">Item<div>Subtitle</div></a></li>
        <li>
            <a href="" class="ng-toolbar-nav-subtitle">Parent<div>Subtitle</div></a>
        </li>
    </ul>
</div>
Content

You can also add custom content to the toolbar, like text, icons, buttons, forms or even a toggle. By default, there is no JavaScript attached to the toggle. As an example, you can add an off-canvas navigation from the Off-canvas component.

Brand content

Add the .ng-toolbar-brand class to an <a> or <div> element to indicate your brand.

Example
Brand
  • Level1 
<div class="ng-toolbar">
    <a class="ng-toolbar-brand" href="#">Brand</a>
    <ul class="ng-toolbar-nav">
        <li class="ng-active">
            <a href=""><i class="ng-icon-home"></i> Level1</a>
        </li>
    </ul>
</div>
Centered content

Add the .ng-toolbar-center class to center your toolbar content or brand. The element needs to be the last child in the toolbar.

Example

  • Item

<div class="ng-toolbar">
    <ul class="ng-toolbar-nav">
        <li class="ng-active"><a href="">Active</a></li>

        <li>
            <a href="">Parent</a>
        </li>

        <ul class="ng-toolbar-nav">
            <li><a href="">Parent</a></li>
            <li class="ng-hidden-small"><a href="">Item</a></li>
            <li class="ng-active"><a href="">Active</a></li>
        </ul>
    </div>
    <div class="ng-toolbar-content ng-toolbar-center">Center</div>
</div>
Custom Content

Add the .ng-toolbar-content class to a <div> element, which serves as a container for your custom content. The content will be centered vertically.

Example
  • Level1
Some link.
Link 
<div class="ng-toolbar">
    <ul class="ng-toolbar-nav">
        <li class="ng-active">
            <a href=""><i class="ng-icon-home"></i> Level1</a>
        </li>
    </ul>
    <div class="ng-toolbar-content">Some <a href="#">link</a>.</div>
    <div class="ng-toolbar-content ng-hidden-small">
        <form class="ng-form ng-margin-remove ng-display-inline-block">
            <input type="text" placeholder="Search">
            <button class="ng-button ng-button-mini ng-button-primary">Submit</button>
        </form>
    </div>
    <div class="ng-toolbar-content ng-toolbar-flip  ng-hidden-small">
        <div class="ng-button-group">
            <a class="ng-button ng-button-mini ng-button-danger" href="#">Link</a>
            <button class="ng-button ng-button-mini ng-button-danger">Button</button>
        </div>
    </div>
</div>
Toggles

Add the .ng-toolbar-toggle class to an <a> or <div> element to create an icon as a toggle.

Add the .ng-toolbar-toggle-alt class to create an alternative icon as a toggle.

Example 
<div class="ng-toolbar">
    <a href="#" class="ng-toolbar-toggle"></a>
    <div class="ng-toolbar-flip">
        <a href="" class="ng-toolbar-toggle ng-toolbar-toggle-alt"></a>
    </div>
</div>
Toolbar flip

Add the .ng-toolbar-flip class to a <div> element to group and align navigations and items to the right.

Example
  • Active
  • Item
  • Parent
  • Parent
  • Item
  • Active
<div class="ng-toolbar">
    <ul class="ng-toolbar-nav">
        <li class="ng-active"><a href="">Active</a></li>
        <li><a href="">Item</a></li>
        <li><a href="">Parent</a></li>
    </ul>
    <div class="ng-toolbar-flip">
        <ul class="ng-toolbar-nav">
            <li><a href="">Parent</a></li>
            <li><a href="">Item</a></li>
            <li class="ng-active"><a href="">Active</a></li>
        </ul>
    </div>
</div>
Toolbar attached modifier

Add the .ng-toolbar-attached class to optimize the toolbar's styling to be attached to the top of the viewport. For example, rounded corners will be removed.

Example
  • Active
  • Item
  • Parent
<div class="ng-toolbar ng-toolbar-attached">
    <ul class="ng-toolbar-nav">
        <li class="ng-active"><a href="">Active</a></li>
        <li><a href="">Item</a></li>
        <li><a href="">Parent</a></li>
    </ul>
</div>
Toolbar with dropdowns

A toolbar can contain a dropdown from the Dropdown component. Just add the .ng-dropdown-toolbar modifier to the dropdown, so it fits perfectly into the toolbar's styling.

Example
  • Active
  • Item
  • Parent
    • Item
    • Another item
    • Header
    • Item
    • Another item
    • Separated item
<div class="ng-toolbar">
    <ul class="ng-toolbar-nav">
        <li class="ng-active"><a href="">Active</a></li>
        <li><a href="">Item</a></li>
        <li class="ng-parent" data-ng-dropdown>
            <a href="">Parent <i class="ng-icon-caret-down"></i></a>
            <div class="ng-dropdown ng-dropdown-toolbar">
                <ul class="ng-nav ng-nav-toolbar">
                    <li><a href="#">Item</a></li>
                    <li><a href="#">Another item</a></li>
                    <li class="ng-nav-header">Header</li>
                    <li><a href="#">Item</a></li>
                    <li><a href="#">Another item</a></li>
                    <li class="ng-nav-divider"></li>
                    <li><a href="#">Separated item</a></li>
                </ul>
            </div>
        </li>
    </ul>
</div>
