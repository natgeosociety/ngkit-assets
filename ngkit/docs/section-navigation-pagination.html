Usage

The Pagination component consists of button-like styled links, that are aligned side by side.

ClassDescription
.ng-pagination Add this class to a <ul> element to define the Pagination component. Use <a> elements as pagination items within the list.
.ng-active Add this class to a list item to apply an active state and use a <span> instead of an <a> element.
.ng-disabled Add this class to a list item to apply a disabled state and use a <span> instead of an <a> element. To apply an ellipsis without any styling, just use a <span> element instead of an <a> element.
Example
  • 1
  • 2
  • 3
  • 4
  • ...
  • 20
<ul class="ng-pagination">
    <li class="ng-disabled"><span><i class="ng-icon-angle-double-left"></i></span></li>
    <li class="ng-active"><span>1</span></li>
    <li><a href="#">2</a></li>
    <li><a href="#">3</a></li>
    <li><a href="#">4</a></li>
    <li><span>...</span></li>
    <li><a href="#">20</a></li>
    <li><a href="#"><i class="ng-icon-angle-double-right"></i></a></li>
</ul>
Previous and next navigation

Creating a simple previous and next pagination is very easy. Just add the .ng-pagination-previous or .ng-pagination-next class to a <li> element to align previous and next buttons left or right.

Example
  • Previous
  • Next 
<ul class="ng-pagination">
    <li class="ng-pagination-previous"><a href="#"><i class="ng-icon-angle-double-left"></i> Previous</a></li>
    <li class="ng-pagination-next"><a href="#">Next <i class="ng-icon-angle-double-right"></i></a></li>
</ul>
