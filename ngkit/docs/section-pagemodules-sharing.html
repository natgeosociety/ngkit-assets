Sharing widget

The sharing widget can vary by size and location. The size options are:

mini.ng-sharing-mini20px high
small.ng-sharing-small30px high
normal.ng-sharing40px high
large.ng-sharing-large50px high

By default the location of the sharing widget is in line with page. Adding .ng-sharing-sticky-left, .ng-sharing-sticky-right or .ng-sharing-sticky-bottom will cause the widget to stick to the appropriate edge of the browser.
