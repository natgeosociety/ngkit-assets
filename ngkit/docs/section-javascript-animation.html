Animation

A basic collection of smooth animations to use within your page.

Animation
Usage
Reverse Modifier
Duration Modifier
Origin Modifier
Animation hover
Usage

To apply this component, add any .ng-animation-* class to an element and it will fade in with a nice animation. These classes are commonly set by using JavaScript to apply the animation to specific behaviors.

Class Description
.ng-animation-fade The element fades in.
.ng-animation-scale-up The element scales up.
.ng-animation-scale-down The element scales down.
.ng-animation-slide-top The element slides in from the top.
.ng-animation-slide-bottom The element slides in from the bottom.
.ng-animation-slide-left The element slides in from the left.
.ng-animation-slide-right The element slides in from the right.
.ng-animation-shake The element shakes.
.ng-animation-scale The element scales down without fading in.

Duration Modifier

To stretch the animation duration to 15 seconds, add the .ng-animation-15 class.

Example
Slide right
<div class="ng-grid ng-text-center">
    <div class="ng-width-medium-1-5 ng-animation-hover">
        <div class="ng-panel ng-panel-box ng-animation-slide-right ng-animation-15">Slide right</div>
    </div>
</div>
Origin Modifier

By default, scaling animations originate from the center. To modify this behavior, add one of these classes.

Class Description
.ng-animation-top-left The animation spreads from the top left.
.ng-animation-top-center The animation spreads from the top center.
.ng-animation-top-right The animation spreads from the top right.
.ng-animation-middle-left The animation spreads from the middle left.
.ng-animation-middle-right The animation spreads from the middle right.
.ng-animation-bottom-left The animation spreads from the bottom left.
.ng-animation-bottom-center The animation spreads from the bottom center.
.ng-animation-bottom-right The animation spreads from the bottom right.

Animation hover

To trigger an animation on hover, just add the .ng-animation-hover class to the animated element itself or to a container.

Example
Fade
Slide right
<div class="ng-grid ng-text-center">
    <div class="ng-width-medium-1-5">
        <div class="ng-panel ng-panel-box ng-animation-fade ng-animation-hover">Fade</div>
    </div>
    <div class="ng-width-medium-1-5">
        <div class="ng-animation-hover">
            <div class="ng-panel ng-panel-box ng-animation-slide-right">Slide right</div>
        </div>
    </div>
</div>
