Usage

The Search component consists of a search form and the search field itself.

Class Description
.ng-search Add this class to a <form> element to define the Search component.
.ng-search-field Add this class to an <input> element to create the search field.
.ng-search-field-rounded Add this class to an <input> element to create the search field with a rounded appearance.

To enable the necessary JavaScript for the search, just add the data-ng-search attribute.

Example 
<form class="ng-search" data-ng-search>
    <input class="ng-search-field" type="search" placeholder="search...">
</form>
<form class="ng-search" data-ng-search>
    <input class="ng-search-field-rounded" type="search" placeholder="search...">
</form>
Ajax Search Results

You can use a dropdown from the Dropdown component to display search results. Just add {source:'PATH/TO/RESULTS'} to the data attribute and set the path to your search results, which need to be formatted in JSON (Example). This enables the Ajax search and injects a dropdown with the .ng-dropdown-search class displaying the search results. You can even navigate through the dropdown with the up and down keys of your keyboard.

IMPORTANT The search requires the autocomplete component to work. Please make sure that you include them always together.

The default renderer takes results like:

{
    "results": [
        {"title":"Article title", "url":"#",
         "text":"Lorem ipsum dolor sit amet, consectetur ..."},
        {"title":"Article title", "url":"#",
         "text":"Ut enim ad minim veniam, quis nostrud ..."},
        {"title":"Article title", "url":"#",
         "text":"Duis aute irure dolor in reprehenderit ..."},
        {"title":"Article title", "url":"#",
         "text":"Excepteur sint occaecat cupidatat non ..."}
    ]
}

And renders them like:

  • Search Results
  • Article title
    Lorem ipsum dolor sit amet, consectetur ...
  • Article title
    Ut enim ad minim veniam, quis nostrud ...
  • Article title
    Duis aute irure dolor in reprehenderit ...
  • Article title
    Excepteur sint occaecat cupidatat non ...
Example 
<form class="ng-search" data-ng-search="{source:'public/sample-search-results.json'}">
    <input class="ng-search-field" type="search" placeholder="search...">
</form>
Search in navbars

A search can be used inside a Navbar from the Navbar component.

Example 
<nav class="ng-navbar">
    <div class="ng-navbar-flip">
        <div class="ng-navbar-content">
            <form class="ng-search" data-ng-search="{flipDropdown:true, source:'public/sample-search-results.json'}">
                <input class="ng-search-field" type="search" placeholder="search...">
            </form>
        </div>
    </div>
</nav>
Search in off-canvas

A search can also be placed inside the off-canvas bar.

Example 
<button class="ng-button" data-ng-offcanvas="{target:'#offcanvas-6'}">Open</button>
<div id="offcanvas-6" class="ng-offcanvas">
    <div class="ng-offcanvas-bar">
        <form class="ng-search">
            <input class="ng-search-field" type="search" placeholder="search...">
        </form>
    </div>
</div>
