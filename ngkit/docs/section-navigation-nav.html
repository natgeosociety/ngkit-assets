Usage

To apply this component, add the .ng-nav class to an <ul> element. Use <a> elements as menu items within the list. To apply an active state to a menu item, just add the .ng-active class.

NOTE By default, the nav has no styling. That's why it is important to add a style modifier class to give it some styling. In our examples we used the .ng-nav-side class.

Example
  • Active
  • Item
  • Item
<div class="ng-width-medium-1-3">
    <ul class="ng-nav ng-nav-side">
        <li class="ng-active"><a href="#">Active</a></li>
        <li><a href="#">Item</a></li>
        <li><a href="#">Item</a></li>
    </ul>
</div>
Nested navs

You can easily add any number of <ul> elements to your nav.

.ng-openAdd this class to the .ng-parent class to expand the parent menu

Accordion

JAVASCRIPT Requires core/nav.js

By default child menu items are always visible. To apply the accordion effect, just add the data-ng-nav attribute to the main <ul>. When clicking on a parent item, an open one will close, allowing only one open nested list at a time. To avoid this behavior, just append {multiple:true} to the data attribute.

Header and divider

Add .ng-nav-header to a list item (<li>) to create a header. Add .ng-nav-divider to a <li> element to create a divider separating menu items.

Class Description
.ng-nav-header Add this class to a <li> element to create a header.
.ng-nav-divider Add this class to a <li> element to create a divider separating menu items.
Example
  • Header
  • Item
    Subtitle
  • Item
  • Item
<ul class="ng-nav ng-nav-side">
    <li class="ng-nav-header">Header</li>
    <li><a href="">Item <div>Subtitle</div></a></li>
    <li><a href="">Item</a></li>
    <li class="ng-nav-divider"></li>
    <li><a href="">Item</a></li>
</ul>
